## State Management

You can pass a reactive value to this Checkbox component, allowing both the child and parent components to use a synchronized value.

```html
<template>
  <div>
    {{ isChecked }}
    <Checkbox v-model:isChecked="isChecked" label="Checkbox" />
  </div>
</template>

<script setup lang="ts">
  import { ref } from 'vue'
  import Checkbox from 'elmethis'

  const isChecked = ref(false)
</script>
```
